# Sumcontrol

Sistema de control de cambios basado en suma SHA256.

Mantiene un histórico de cambios con las sumas, en el archivo generado `sumcontrol_history`.

Por defecto, el directorio sobre el que trabajará será `files`.

```
workdir/
├── files/
│   ├── file1
│   ├── file2
│   └── file3
└── sumcontrol
```
